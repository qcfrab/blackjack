# Blackjack Telegram Bot

Реализация Блэкджека на Ruby. Проект переведён с консольного интерфейса на Telegram-бота.

## Установка и запуск
1. **Клонирование:** `git clone https://github.com/qcfrab/blackjack.git && cd blackjack`
2. **Зависимости:** `bundle install`
3. **Конфигурация:** Создайте `.env` и добавьте `TELEGRAM_BOT_TOKEN=ваш_токен`
4. **Запуск:** `ruby bin/bot.rb`

## Использование
* **Старт:** Нажмите `/start`, бот инициализирует банкролл ($1000).
* **Ставка:** Введите любое число в чат (например, `50`).
* **Ход:** Используйте кнопки **«Взять ещё»** или **«Хватит»** под сообщением с картами.
* **Настройка:** Кнопка **«Сменить стратегию»** открывает меню выбора стратегии игры для дилера.
* **Тесты:** Запуск через `ruby -Ilib:test test/test_helper.rb`.
